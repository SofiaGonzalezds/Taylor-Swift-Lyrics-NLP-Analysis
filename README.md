# Taylor Swift Lyrics NLP Analysis

This repository presents a Natural Language Processing project focused on exploring thematic and stylistic patterns in Taylor Swift’s lyrics. The project analyzes lyrics from her first nine albums and applies different NLP techniques to understand recurring themes, lyrical evolution, album cohesion, and text generation possibilities.

## Repository Contents

- **eda.ipynb**: Notebook containing the exploratory data analysis of the lyrics dataset, including song length analysis, word frequency, and sentiment exploration.

- **word_embeddings.ipynb**: Notebook used to create and analyze word embeddings with Word2Vec, exploring how words are represented across the corpus and albums.

- **clustering.ipynb**: Notebook focused on clustering and topic modeling techniques to identify common lyrical patterns and thematic groups.

- **gpt2_text_generation.ipynb**: Notebook used to experiment with GPT-2 for generating new lyrics based on Taylor Swift songs.

- **final_presentation.pdf**: Final presentation summarizing the methodology, findings, limitations, and conclusions of the project.

- **project_proposal.docx**: Initial project document describing the corpus, motivation, research questions, and proposed methodology.
